## 学习建议

1. **新手入门**：建议从 01-05 开始，了解基础功能实现
2. **3D 开发**：学习 06-11，掌握 Three.js 相关知识
3. **后端开发**：学习 12-17，理解后端架构设计
4. **UI 开发**：学习 18，掌握 Element Plus 组件使用
5. **业务功能**：学习 20-22，理解店铺、商品和用户管理流程 ⭐ 更新
